# Real-Time Chat Application Using AWS

## Overview

This project is a serverless real-time chat application built using:

- AWS Lambda
- Amazon API Gateway (WebSocket API)
- Amazon DynamoDB
- HTML
- CSS
- JavaScript

## Features

- Real-time messaging
- Serverless backend
- Message storage
- WebSocket communication
